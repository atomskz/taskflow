## Дальнейшие идеи

Необязательные улучшения, не входившие в исходный список.

### IDEA-001: Серверные данные для календаря и командной палитры
Сейчас календарь и command palette всё ещё грузят полный список задач (`listTasks`, до 500). Для масштаба — отдавать задачи по диапазону дат (`from`/`to`) для месяца календаря и серверный поиск для палитры, чтобы полностью убрать загрузку всего списка на клиент.

### IDEA-002: Полное покрытие тёмной темы
Несколько декоративных градиентов на лендинге/странице входа намеренно остаются тёмно-цветными. Стоит пройтись по ним прицельно, если нужна 100% консистентность.

### IDEA-003: Rate limiter для горизонтального масштабирования
Текущий лимитер — in-memory (один процесс). Для нескольких инстансов вынести в общий стор (Redis или таблица в БД).

### IDEA-004: E2E в CI
Подключить Playwright в CI (GitHub Actions): кэш браузеров + `npx playwright install`, прогон `npm run test:e2e` на pull request.

### IDEA-005: Очистка корзины
Авто-удаление мягко удалённых задач старше N дней (cron/при старте) и кнопка «Очистить корзину» в UI.

### IDEA-006: Безопасность сессий
Список активных сессий пользователя и «выйти на всех устройствах» (массовый отзыв refresh-токенов); опционально — повторный вход по email-коду.
